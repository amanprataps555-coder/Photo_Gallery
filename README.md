# Best Friend Photo Gallery

A simple website showcasing photos with my best friend, organized into different sections.

## Sections
- Memories
- Adventures
- Fun Moments
- Special Days
- Music
- Fun Zone (with a quiz game)

## How to View
Open `index.html` in your web browser.

## Customization
- Replace the placeholder image URLs in `index.html` with your actual photo URLs or local paths.
- Add your music files to the `music` folder and update the audio sources.
- Customize the quiz questions and answers in `script.js` to match your friendship.

## Features
- Responsive design
- Lightbox for viewing images
- Smooth hover effects
- Audio players for music
- Interactive quiz game
